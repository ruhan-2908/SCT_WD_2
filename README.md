# DOM Calculator

This is a fully functional calculator built using **HTML, CSS, and JavaScript**. It was created to practice **DOM manipulation**, **event handling**, **keyboard support**, and **basic arithmetic operations** as part of an internship task.

## Features

-  Real-time expression building
-  Keyboard input support
-  Button click handling
-  Basic error handling (invalid expressions)
-  Backspace (`←`) and Clear (`C`) support
-  Stylish and responsive user interface
-  Built without any libraries or frameworks

##  Technologies Used

- HTML
- CSS
- JavaScript (DOM + Events + `eval()`)

##  Keyboard Shortcuts

| Key           | Action             |
|---------------|--------------------|
| Numbers/`+-*/`| Input              |
| `Enter`       | Evaluate expression|
| `Backspace`   | Delete last char   |
| `Escape`      | Clear all          |
